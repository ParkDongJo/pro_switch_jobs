# 접근성 트리

웹브라우저에서 접근성 탭을 누르면, 웹사이트의 전체 접근성 tree 를 시각적으로 보여준다. 이 접근성 tree 에서

주요한 상태

- 역할
- 이름
- 설명

그 외 상태

- 포커스 가능
- 값 설정
- 읽기전용
- 수준

상태들이 있다. 그 중 주요한 상태들을 더 살펴보면

## 역할

시멘틱 태그는 기본 역할을 가진 태그들 - input, button 등등
역할이 없는 태그 - div

역할이 없는 테그는 기본적으로 generic 역할로 처리된다.
역할을 강제로 부여하고자 한다면 WAI-ARIA 보조기술을 활용해서, aria-role 을 부여한다.

## 이름

각 요소의 텍스트 정보를 이름이라고 한다. 접근 가능한 이름을 통해 사용자는 특정 요소가 어떤 기능을 수행하는지 힌트를 얻을 수 있다.

텍스트 정보

- p, span, button, label 등에 들어가는 텍스트
- img 의 alt 속성에 들어가는 텍스트
- title 속성
- aria-label, aria-labeledby속성에 들어가는 텍스트

Presetational Children
특정 role 을 가진 요소들은 태그 내부에 있는 자식요소들에서 접근가능한 이름들을 모아서 contents 로 사용한다.
