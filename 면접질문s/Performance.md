

## 웹페이지 성능 최적화를 한다면 어떻게 할건가요
-----
일단 프론트에서 웹서비스에서 성능 측정의 구분을 크게 2개로 봐야합니다.

- 로딩 성능
	- 서버에 있는 웹 리소스를 다운로드 할 때의 성능을 말합니다.
		- 리로스 크기, 개수 최적화
		- 코드 분할
		- 리소스 우선순위
- 랜더링 성능
	- 다운로드한 리소스를 가지고 화면을 그릴 때의 성능을 말합니다.
		- 브라우저 동작원리
		- 프레임워크 라이프 사이클

만약 웹서비스 전체로 파악한다면 더 다양한 영역들이 있습니다. 너무 깊게 가진않고 나열만 해볼게요.
