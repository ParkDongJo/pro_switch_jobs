# 반복문

for
```kotlin
for (i in 0..3) {
   print(i)
}
// 0 1 2 3
```

reverse for
```kotlin
for (i in 3 downTo 0) {
   print(i)
}
// 3 2 1 0
```


step for
```kotlin
for(i in 0..5 step 2){ 
	print(i)
}
// 0 2 4
```
