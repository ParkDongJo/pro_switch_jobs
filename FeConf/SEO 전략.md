웹퍼널
- 검색노출 - 브랜딩 가치 유입 가능성 증가
- 검색유입 - 비용 / 고객수 = CAC(고객객단가) 낮춤
- 이탈율 / 전환율 - 전환 고객수 = LTV(고객 1명당 비즈니스 가치) 증가


전략 1.  페이지 노출 많이 하기
수집 단계
발견 안됨 -> 발견 -> 크롤링됨 -> 인덱싱됨

발견이 되게 하려면
button -> a
sitemap & robot.txt 제공
CSP -> SSR

크롤링 되게 하려면
- 크롤링 버짓 관리 -> 버짓이란 로봇이 하루에 우리 웹사이트에 쓸수있는 시간의 총량
	- 페이지 수집에 걸리는 시간을 줄인다
	- LCP 개선하는 것이 중요하다
- 의도적 noIndex
- canonical 설정

인덱스 생성요청
- 수동...
- 양이 제한됨

디테일한 전략 목표
버짓을 동안 머무는 시간에 최대한 많은 정보를 제공 -> LCP 개선
버짓을 키운다
- 로봇이 머무는 시간을 늘린다.


노출 페이지 증가
- 노출될 정보의 조합으로 노출되는 페이지 선정 후 배포
- sitemap 으로 최대한 주요 페이지 제공
