# 조건에 따라 다른 JSX를 반환하는 방법
3가지 방법

#### if~else
JSX return 를 if ~ else 로 분기할 수 있다.
이를 활용해 빈 `<></>` 를 return 하거나 null 을 리턴할 수 있다.

#### 삼항 연산자 ? :
코드의 중복이 발생 시, 그 대안으로 삼항연산자를 소개

```
if (isPacked) {  

return <li className="item">{name} ✅</li>;  

}  

return <li className="item">{name}</li>;
```

```
return (  

<li className="item">  

{isPacked ? name + ' ✅' : name}  

</li>  

);
```

이 스타일은 간단한 조건에 잘 어울리지만, 적당히 사용하는 게 좋습니다

[**중요한 부분**]
중첩된 조건부 마크업이 너무 많아 컴포넌트가 지저분해질 경우 자식 컴포넌트를 추출하여 정리하세요
