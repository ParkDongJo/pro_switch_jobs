텍스트 스타일링 실사용 예시 모음

1. 줄바꿈 (white-space, word-break, overflow-wrap)

같은 텍스트를 줄바꿈 규칙만 다르게 적용한 예시입니다.

white-space: normal (기본) This is a long text with many spaces. 안녕하세요 이것은 아주아주아주아주긴텍스트입니다1234567890
white-space: nowrap This is a long text with many spaces. 안녕하세요 이것은 아주아주아주아주긴텍스트입니다1234567890
white-space: pre This is a long text with many spaces. 줄바꿈 띄워쓰기도 그대로 유지합니다.
white-space: pre-wrap This is a long text with many spaces. 줄바꿈도 유지하면서 자동 줄바꿈도 됩니다.
word-break: break-all URL: https://www.example-website.com/very/long/path/to/resource 긴영어단어 supercalifragilisticexpialidocious 도중에도 끊김
overflow-wrap: break-word URL: https://www.example-website.com/very/long/path/to/resource 긴영어단어 supercalifragilisticexpialidocious 는 단어 단위로 줄바꿈

2. 말줄임 (ellipsis)

카드 제목, 리스트 등에서 자주 쓰는 말줄임 예시입니다.

한 줄 말줄임 이것은 한 줄 말줄임 예시입니다. 길어지면 이렇게 ... 으로 잘립니다.
여러 줄 말줄임 (3줄) 이것은 여러 줄 말줄임 예시입니다. 텍스트가 길어질 경우 세 줄까지만 보여주고 그 이후 내용은 숨겨집니다. 카드 설명, 게시글 미리보기, 알림 메시지 요약 등에 자주 사용됩니다. 이 문장은 실제로는 보이지 않을 수 있습니다.

3. 간격 (letter-spacing, line-height, word-spacing, 문단 간격)

letter-spacing: -0.05em (자간 좁게) 자간을 약간 좁게 설정하면 헤드라인 같은 곳에서 타이트한 느낌을 줄 수 있습니다.
letter-spacing: 0.15em (자간 넓게) 넓은 자간은 로고 타입이나 버튼 레이블에 자주 사용됩니다.
line-height: 1.1 (행간 좁게) 행간이 좁은 텍스트입니다. 여러 줄로 작성했을 때 꽉꽉 붙어 보입니다.
line-height: 1.8 (행간 넉넉) 행간을 넉넉하게 주면 긴 본문 텍스트의 가독성이 좋아집니다. 블로그 글, 설명문, 문서 본문 등에 추천되는 스타일입니다.
word-spacing: 0.4em This sentence has wider word spacing to make each word stand out a bit more.
문단 간 간격 (margin-bottom)

첫 번째 문단입니다. 아래쪽에 여백이 살짝 들어갑니다.

두 번째 문단입니다. 문단 사이가 붙지 않아 읽기 좋게 보입니다.

4. 정렬 (text-align)

text-align: left 왼쪽 정렬은 기본적인 텍스트 정렬 방식으로, 대부분의 본문에서 사용됩니다.
text-align: center 가운데 정렬은 제목, 버튼 라벨, 카드 헤더 등에 자주 사용됩니다.
text-align: right 오른쪽 정렬은 날짜, 가격, 메타 정보 정렬 등에 사용하기 좋습니다.
text-align: justify (양쪽 정렬) 양쪽 정렬은 신문, 잡지 같은 느낌을 줄 수 있지만, 한글에서는 단어 간 간격이 부자연스럽게 보일 수 있습니다.

5. 텍스트 장식 (text-decoration)

밑줄 이 텍스트는 밑줄이 그어진 상태입니다.
취소선(line-through) 29,000원 39,000원
overline 텍스트 위쪽에 선이 표시됩니다.
wavy underline + color 텍스트 장식에 물결 효과와 색을 줄 수 있습니다.
링크 밑줄 제어 이 링크는 기본적으로 밑줄이 없고,
hover 시에만 밑줄이 생깁니다.

6. 텍스트 변환 (text-transform)

uppercase button primary
lowercase WARNING MESSAGE
capitalize user profile settings page

7. 텍스트 그림자 (text-shadow)

soft shadow 부드러운 텍스트 그림자 효과입니다.
strong shadow 강한 텍스트 그림자 효과입니다.

8. 실전 예시: 카드 레이아웃에서의 텍스트 스타일링

실제로 서비스에서 많이 쓰는 “카드 목록” 예시입니다. 제목/설명/메타 정보에 각각 다른 텍스트 스타일이 적용되어 있어요.

notice
긴 제목이 들어가는 카드 컴포넌트 예시입니다. 두 줄까지만 표시되고 그 이후는 잘립니다.
이 영역은 카드의 설명 텍스트입니다. 최대 세 줄까지 보여주고 나머지는 숨겨집니다. 실제 서비스에서는 공지사항, 게시물 요약, 상품 설명 등에 활용할 수 있습니다.
update
프로필 설정 기능이 새로 추가되었습니다.
마이페이지 > 계정 설정에서 프로필 사진, 닉네임, 상태 메시지를 변경할 수 있습니다. 모바일 환경에서도 동일하게 사용할 수 있도록 반응형으로 적용되었습니다.
blog
텍스트 스타일링만 잘해도 서비스 퀄리티가 한 단계 올라가는 이유
타이포그래피는 UI 디자인에서 가장 중요한 요소 중 하나입니다. 글자 크기, 간격, 줄 수 제한, 정렬만 잘 조합해도 사용자 경험이 크게 개선됩니다.
