# 필요 사항

키클락 기반 IAM 의 인증 화면에서 모바일 환경의 웹접근성 이슈가 나왔었다. 수정은 했지만 실제 모바일 환경에서만 확인 가능한 사항이였기 때문에 PC 에서 띄운 로컬 서버에 모바일에서 접근이 가능해야 했다.

# 문제점

처음에는 IP 기반으로 접근을 하려고 했지만, 여러가지 이유로 막혔다.

- HTTPS 로 접근을 해야한다는 점
- VPN 으로 내부망으로 접근해야한다는 점

# 해결방안 찾기

일단 내부망이야 VPN을 모바일에 설치해서 접근하면 됐다.

문제는 HTTPS 프로토콜로 접속해야 인데, 두가지 방향이 있었다.

- Proxy 서버를 올려서 HTTPS 로 접근할 수 있도록 한다.
- Keycloack 서버 내에서 설정할 수 있는 걸 찾아본다.

ngrok 이라는 proxy 서버를 활용
