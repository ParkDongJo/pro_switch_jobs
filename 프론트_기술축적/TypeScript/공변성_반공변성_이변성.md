
## 공변성
좁은 타입 변수에 넓은 타입 변수 할당이 가능하다.

예시 상황
- 함수의 리턴값
- 일반 변수

```typescript
let users: Array<User> = []
let members: Array<Member> = []

users = members;
members = users; // ERROR!!!

let a = (id: string): User => ({ id });
let b = (id: string): Member => ({ id, nickname });

a = b;
b = a; // ERROR!!!
```

## 반공변성
넓은 타입 변수에 좁은 타입 변수 할당이 가능하다.

예시 상황
- 함수의 매개변수
- 제네릭

```typescript
let a = (param: User): string => param.id;
let b = (param: Member): string => param.id

a = b; // ERROR!!!
b = a;

const user: User = { id: "123" };
let a = <T extends User>(param: string): T => user as T
let a = <T extends Member>(param: string): T => user as T

a = b; // ERROR!!!
b = a;
```

## 이변성
좁은 타입 변수에 넓은 타입 변수 할당도 가능하고, 넓은 타입 변수에 좁은 타입 변수 할당도 가능하다.
