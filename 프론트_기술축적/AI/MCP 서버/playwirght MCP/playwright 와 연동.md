# Playwright MCP 커스텀 해보기

주요 내용의 영상 흐름

브라우저 언더바 클릭 도구는 한 번에 하나의 요소만 클릭할 수 있어, MBTI 테스트와 같이 많은 질문에 일일이 답해야 하는 경우 비효율적이다.

여러 요소를 동시에 클릭할 수 있는 브라우저 언더바 클릭 언더바 멀티플(Click_Multiple)과 같은 도구가 필요하지만, 기본 MCP 서버에는 해당 기능이 없다.

GitHub에서 MCP 서버 소스 코드를 다운로드 받아 커서 에디터로 열고,

- 커서 AI에게 프로젝트 구조와 기능을 파악하도록 요청한다.
- 사용자는 커서 AI에게 브라우저 언더바 클릭 멀티플 도구를 추가해 달라고 요청하며, 여러 요소를 0.3초~3초 간격으로 클릭하는 기능을 구현하도록 한다.
- 커서 AI가 만든 코드를 수락한다.
- 빌드(build) 명령어를 실행해 실행 파일을 생성한다.
- 커서 AI 에게 프로젝트 실행방법을 알려달라고 한다.
- 실행 방법을 커서 AI MCP 설정에서 command와 args 에 반영한다.
- 로컬의 MCP 코드를 바탕으로 MCP 서버를 실행하여, 커서 AI 가 활용 할 수 있도록 한다.
