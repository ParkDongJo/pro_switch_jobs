
## Story

각 컴포넌트에 대한 Story 를 만들수 있고 확장자는 Component.stories.jsx 가 된다. 컴포넌트를 아래와 같이 Story 에 등록을 해야한다.

## 설정
스토리북 기본 설정들은 아래 .storybook 디렉토리에서 설정하면된다. 그 중 preview.js 는 스토리분 전역 설정이다.

```
.storybook/
  main.js     <-- addOn 설정, 빌드 설정
  preview.js  <-- Global 설정
  ...
```

스토리북의 설정은 반영이 아래와 같은 상속 관계를 가진다.

> Global 단계 -> Component 단계 -> Story 단계
> preview.js  ->  .storeis.jsx          -> .stories.jsx
>           -> export default     -> export

### addon-controls
props 전달을 확인해 볼 수 있도록 돕는다.
기본적으로 제공되는 애드온이다. @storybook/addon-essentials 에 포함된 애드온이다.

### addon-actions
이벤트 핸들러를 테스트 해볼 수 있도록 돕는다.
기본적으로 제공되는 애드온이다. @storybook/addon-essentials 에 포함된 애드온이다.

### addon-viewport

### msw-storybook-addon

### addon-interactions
