
간단 결론

SDK 와 슈퍼앱 간의 진입 화면을 만든다. 진입화면에서 Webview 를 load 시킨다.

진입화면에서 Controller 를 떼어내고, 이 컨트롤러에서는 Native 를 제어할 기능 코드들을 넣는다.
진입화면에서 View 는 view 담당 xml 이나 swift UI와 연결 시킨다

Bottom Tab 을 Native로 할 지, Web 으로 할 지는 좀 더 고민해보기.

로그인이 중간 시점에 필요할 시, 슈퍼앱의 로그인 화면으로 왔다 갔다 할때 Deeplink 로 가야할 수도 있다. (좀더 알아보기)

알림 목록을 간다고 했을때도 마찬가지, 하지만 예시로 TMAP 을 봤을 때는 SDK 쪽에서는 알림목록 진입자체가 불가능하게 되어 있다.

마이페이지는 SDK 내부에 별도로 또 가진다.

만약 외부에서 특정화면이 필요하다고 하다면, 각 특정화면에 대한 진입화면을 SDK에서 만들어준다.
