# closest 로 부모를 찾자
`closest()`는 웹 브라우저의 DOM API에서 제공하는 메서드이다. 자기 자신을 포함하여 문서 트리를 위쪽으로 순회하면서 주어진 CSS 선택자와 일치하는 가장 가까운 첫번째 조상 요소를 찾는다.

아래와 같이 사용이 가능하다.

```jsx
// DOM 구조가 있다고 가정
// <div>
//   <ul class="list">
//     <li>Item 1</li>
//     <li>Item 2</li>
//   </ul>
// </div>

const element = document.querySelector('li'); // 특정 li 요소를 선택
const closestUl = element.closest('.list'); // 선택된 li에서 가장 가까운 .list 요소를 찾습니다.
console.log(closestUl); // <ul class="list">...</ul>이 출력될 것입니다.

```


## 그 반대 개념은?
`closest()` 함수의 반대 개념은 가장 가까운 하위 자식 요소를 찾는 것입니다.  
이에 해당하는 함수는 DOM 트리를 아래로 탐색하는 것이며, querySelector 함수를 사용한다.

```javascript
// 특정 요소 내에서 selector 에 해당하는 가장 첫번째 자식 찾기
const regionElement = screen.getByRole('region', { name: '주요 내용' });

regionElement.querySelector('.item')
```
